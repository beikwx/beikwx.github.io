## Rule 1.Linear Carboxylic acid

> 1.Select longest carbon chain starting from COOH, number the chain starting from COOH. Write name by replacing last 'e' by 'oic acid'.

1.从COOH开始选择最长碳链，书写时替换结尾的'e'成
oic acid'

![酸命名][01]

> 2.Write names of substituents in alphabetically order.

2.取代根据字母顺序书写。

![取代][02]

> 3.start number from COOH, it may be anywhere in chain.

3.从COOH开始编号，不管在哪里。

![编号][03]

## Rule 2.Poly caroxylic acids

> 1.If two COOH are present, select chain starting from one COOH and ending with second COOH. Start numbering from that COOH, from where other substituents gets minimum number. Write name by retaining last 'e' following by 'dioic acid'.

1.如果有两个酸，从一个酸到另一个酸进行编号。命名时保留结尾'e'加入'dioic acid'.

![二酸][14]

> 3.If three or more COOH are present, select chain to which all of them are attached, exclude COOH from chain. Write name by adding carboxylic acid to name of alkane. Last 'e' of alkane is retained.

3.如果有3个以上的酸，选择包含所有酸的碳链作为母链。书写时在烷烃后面加上'carboxylic acid'，烷烃结尾的'e'进行保留。

![多酸][15]

## Rule 3.Cyclic carboxylic acid

> 1.If COOH is present on ring, number the ring starting from corner where COOH is attached. Write name by adding carboxylic acid to name of ring. Retain last 'e' of ring.

1.如果COOH在环上，从COOH连接的碳开始编号。命名时在环烷烃后面加上'carboxylic acid'。保留环烷烃结尾的'e'。

![环酸][24]

> 2.Benzene carboxylic acid is called benzoic acid, if other substituents are present, name is written by writing benzoic acid as parent.

2.苯环上的酸叫做苯甲酸。如果还有其他取代，把苯甲酸作为母体。

![苯甲酸][25]

## Rule 4.Poly cyclic acids

> 1.If two or more COOH are present on ring, start numbering from any one COOH so that sum of locants is low.

1.如果环上有多个COOH，从任意COOH开始编号让位置总和最低。

![环多酸][34]

> 2.Write name by adding caoboxylic acid to name of ring.

2.书写时在环结尾加上酸。

![书写][35]

> 3.According to number of COOH groups, prefixeds like di, tri etc are used.

3.多酸使用di,tri等前缀。

![多酸][36]

## Rule 5.Unsaturated acids

> 1.In case of double or triple bond containing chains, number the chain from COOH and write name by replacing last 'e' by 'oic acid'.

1.如果有双键或者三键，从COOH开始编号，替换结尾'e'成'oic acid'。

![不饱和键][45]

> 2.If ring containg double bond, retain last 'e' of corresponding cycloalkene and add 'carboxylic acid'。

2.如果环含有双键或者三键，保留环烃的'e’结尾加上'carboxylic acid'。

![环烯酸][46]
