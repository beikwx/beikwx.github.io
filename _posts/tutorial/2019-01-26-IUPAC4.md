## Rule 1.Ether 1

> 1.Ether are named as alkoxy alkane. If alkyl groups are methyl or ethyl, numbering is not required.

1.醚根据氧基烷烃来命名。如果烷烃是甲基或者甲基，不需要编号。

![醚][01]

> 2.If one or both alkly groups have more than 2 carbons - select group having longest carbon chain as parent, number the chain so that alkoxy(RO) group gets minimum number.

2.如果烷烃有2个以上的碳，选择母链最长的作为烷烃，烷氧基编号最低。

![大于2个][02]

> 3.If another group is present on parent chain besides alkoxy group, write names in alphabetically order. eg. methoxy is written before methyl.

3.如果母链除了烷氧基还有别的基团，按照字母顺序书写。比如甲氧基(methoxy)在甲基(methyl)的前面。

![除了烷氧基][03]

## Rule 2.Ether 2

> 1.If another group is present on parent chain besides alkoxy group, number the chain using 1st point of difference rule, keeping alphabetically order in mind, if two sets of locants are same.

1.如果母链除了烷氧基，用优先规则，如果多个位置一样，让字母顺序靠前的先写。

![多个取代][08]

> 2.If alkoxy group is complex - number the longest chain in alkoxy group, note the number of carbon attached to O, if other substituents are present, note their number, write name as follow: (name of substituents name of parent alkane replacing last e by (locant of C bonded to O)-yl)oxy.

2.如果烷氧基十分复杂：将烷氧基最长链编号，注意和氧原子相连的碳的位置，如果还有其他取代，注意编号，按照如下方法书写：取代母链的名字结尾去掉e变成（碳氧位置-yl)oxy。见下面例子。

![复杂烷氧基][09]

> 3.While naming alkoxy groups, common terms as iso-propyl, tert-butyl etc can be used.

3.命名烷氧基是，普通命名的比如异丙基和叔丁基等以可使用。

![普通命名][10]

## Rule 3.Ether 3

> 1.If only alkoxy group is present on ring, numbering is not required.

1.如果环烷烃上只有一个烷氧基，不需要编号。

![不需要编号][17]

> 2.If alkoxy group is cycloalkyl, it's name is written as cycloalkyloxy. eg. cyclopentyloxy, cyclobutyloxy etc.

2.如果烷氧基是环烷烃，写作环烷烃氧。见例子。

![环烷烃][18]

> 3.If other groups are present on ring besides alkoxy, number the ring so that sum of locants is lower.

3.如果环上还有别的基团，编号让总和最小。

![多个][19]

## Rule 4.Ether with benzene(苯醚)

> 1.Methoxy group attached to benzene is called anisole.

1.甲基直接和苯连接叫做苯甲醚。

![苯甲醚][26]

> 2.If one other group is present besides alkoxy, number the benzene ring so that alphabetically priority group gets lower number.

2.如果除了烷氧基还有别的基团，让字母顺序最低。

![多个][27]

> 3.If two or more other groups are present besides alkoxy, number the ring so that sum of lacants is lowers.

3.多取代编号最低。

![多个][28]

## Rule 5.Poly ethers(多醚)

> 1.If more than one alkoxy groups are present, select long chain to which all of them are attached as parent alkane.

1.如果有多个烷氧基，选择包含有所烷氧基的最长母链。

![多个烷氧基][35]

> 2.If two different sets of locants are possible, use 1 st point of difference rule.

2.如果有两种可能位置，使用第一原则。

![第一原则][36]

> 3.In case of more than one alkoxy groups present on benzene or cycloalkane rings, number the ring to get lower sum of locants.

3.如果在苯环或者环烷烃上有多个烷氧基取代，编号让他们总和最低。

![总和最低][37]
